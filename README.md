# MarkdownViewer

Mac専用の非常にシンプルで軽量なマークダウン（`.md`）ファイルビュワーです。

## 特徴
- **ドラッグ＆ドロップ対応**: アプリウィンドウにマークダウンファイルをドラッグ＆ドロップするだけで、書式付きでレンダリングされます。
- **ダークモード専用**: 目に優しい暗めの背景色に固定されています。
- **読みやすいレイアウト**: 幅いっぱいに広がりすぎないよう適度な余白（2rem）が設定されており、長文も快適に読めます。
- **閲覧専用**: 意図しない編集を防ぐための閲覧（プレビュー）専用アプリです。

## インストール・実行方法

### 1. アプリ単体での実行 (推奨)

パッケージ化された `.app` ファイルをそのまま開くことで実行できます。

1. プロジェクト内にある `MarkdownViewer.app` を見つけます。
   （例: `./MarkdownViewer.app`）
2. Finderからダブルクリックで起動します。
   ※ 必要に応じて、Macの「アプリケーション（Applications）」フォルダに移動させてお使いください。

### 2. ソースコードからのビルド・実行

開発やカスタマイズを行う場合は、プロジェクトルートでSwift Package Manager経由で実行します。

```bash
cd /path/to/MarkdownViewer
swift run
```

## 使い方
1. アプリを起動します。
2. 『Drag and drop a **Markdown** file here.』という画面が表示されます。
3. 表示させたい `.md` 拡張子のファイルを、Mac上のFinderなどからアプリのウィンドウ内にドラッグ＆ドロップします。
4. ファイルの内容がマークダウンとしてスタイリングされて表示されます。

## 開発環境・利用ライブラリ
- 言語: Swift (SwiftUI)
- 対象OS: macOS 13.0 以降
- マークダウンレンダリング: [swift-markdown-ui](https://github.com/gonzalezreal/swift-markdown-ui)
